News site на Laravel 5.8

.env :
DB_DATABASE=news
DB_USERNAME=root
DB_PASSWORD=

Новостной сайт, с выводом всех новостей и выводом новостей по категориям.

news/admin - админ панель для добавления и редактирования категорий и новостей.


news.sql - тестовая БД.
Добавлен seeds только для новостей.
